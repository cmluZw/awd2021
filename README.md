在去年制作的awd-lastest的基础上，今年采用了tp6进行创作，旨在学习tp6

##目录
* admin
    * controller
    * docker
    * model
    * validate
    * view
* common
    * controller
    * model
    * validate
    * view
   
   

##环境
> 运行环境要求PHP7.1+，兼容PHP8.0

> 关于本项目中用到的awd-lastest镜像是一个有lamp环境的镜像，可自行搭建，如有需要可留言

> 要求安装docker


~~~
   ape-get install docker
~~~
